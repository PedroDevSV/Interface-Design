Glosario de Bootstrap 5

Filas y Columnas

Bootstrap 5 utiliza un sistema de rejilla (grid system) basado en filas (.row) y columnas (.col).

Filas

.row: Contenedor de fila que agrupa columnas.

Columnas

.col-6: Columna que ocupa 6 partes del total de 12.

.col-6: Otra columna que ocupa 6 partes del total de 12.

Ejemplo de Rejilla

.col-4
.col-4
.col-4

Tamaños de Pantalla

Bootstrap 5 permite definir tamaños de columna para diferentes tamaños de pantalla usando clases específicas.

Tamaños de Pantalla

  • .col-sm-: Pequeño (≥576px)
  • .col-md-: Mediano (≥768px)
  • .col-lg-: Grande (≥992px)
  • .col-xl-: Extra Grande (≥1200px)
  • .col-xxl-: Extra Extra Grande (≥1400px)

Ejemplo de Tamaños de Columna

.col-sm-6 .col-md-4 .col-lg-3 .col-xl-2 .col-xxl-1
.col-sm-6 .col-md-4 .col-lg-3 .col-xl-2 .col-xxl-1

Offset (Desplazamiento de Columnas)

Bootstrap permite desplazar columnas a la derecha usando clases de offset.

Ejemplo de Offset

.col-4 .offset-2
.col-4 .offset-1

Espacio entre Columnas con Justify-Content

Para dejar espacio entre columnas, se puede utilizar la clase justify-content.

Ejemplo de Justify-Content

.col-4
.col-4
.col-4

Order (Orden de Columnas)

La clase order permite cambiar el orden de las columnas.
